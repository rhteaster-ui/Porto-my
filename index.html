System Profile

Not a template bio. Ini profil operator web yang tumbuh dari keterbatasan.

profile.readme

Saya adalah pengembang web mandiri yang berfokus pada web app, PWA, dan website statis interaktif dengan pendekatan efisien, ringan, dan fungsional.

Perjalanan saya dibangun secara otodidak, tanpa mentor langsung, dengan perangkat terbatas. Dari proses itu, saya terbiasa mendesain, membangun, debug, refactor, optimasi, dan deploy secara iteratif.

AI saya gunakan sebagai productivity tool untuk mempercepat pekerjaan, bukan sebagai pengganti pemahaman. Fokus utama saya adalah membuat web yang benar-benar berguna, stabil, dan nyaman dipakai pengguna umum.
